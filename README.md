#  University Query Analysis (EDA Project)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the University Query dataset.
The goal is to understand patterns, relationships, and important insights from the data.

---

##  Objectives

* Clean the dataset (handle missing values & duplicates)
* Perform summary statistics
* Analyze numerical and categorical features
* Create visualizations
* Categorize deadline levels (Low, Medium, High)

---

##  Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Information

The dataset contains details such as:

* Days_To_Deadline
* Student Queries
* Response Time
* Other academic-related features

---

## 🔍 Steps Performed

### 1️⃣ Data Cleaning

* Checked missing values
* Removed duplicates
* Corrected inconsistent data

### 2️⃣ Feature Engineering

Created a new column to categorize deadlines:

* 0–5 days → Low
* 6–10 days → Medium
* Above 10 days → High

### 3️⃣ Data Visualization

* Heatmap (Correlation Matrix)
* Pair Plot
* Box Plot
* Scatter Plot

---

## 📈 Key Insights

* Identified relationship between deadline and response time.
* Found patterns in student query behavior.
* Categorized deadlines into meaningful groups.
